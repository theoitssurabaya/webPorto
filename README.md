This portfolio is designed to provide a comprehensive view of my skills and achievements. It features a modern, responsive, and dynamic user interface with smooth animations and interactive elements to ensure an engaging user experience.

## ✨ Features

- **Modern & Dynamic UI/UX:** A vibrant light mode aesthetic with animated gradient backgrounds, glassmorphism elements, and smooth scrolling.
- **Interactive Elements:** Custom cursor, hover effects, and a tech stack ticker to bring the design to life.
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile viewing.
- **Structured Sections:**
  - **Profile:** An introduction to my background and core interests.
  - **Skills:** Categorized technical competencies in AI/ML, Embedded Systems/IoT, and Software Engineering.
  - **Education:** My academic timeline at Institut Teknologi Sepuluh Nopember (ITS).
  - **Experience & Certifications:** A timeline of my professional roles, leadership positions, and industry certifications.
  - **Projects:** Featured engineering works linking to respective repositories.
  - **Contact:** Links to my GitHub, LinkedIn, and email for easy communication.

## 🛠️ Technologies Used

- **Next.js 16 (App Router):** Core framework for SSR and routing.
- **React 19:** Component-based UI architecture.
- **CSS Modules / Global CSS:** Custom styling, glassmorphism UI, and responsive layouts.
- **AOS (Animate On Scroll):** Library used for scroll-triggered animations.
- **next/font/google:** utilizing 'Outfit' for headings and 'Inter' for body text.
- **next/image:** Automatic WebP image optimization and lazy loading.

## 🚀 Getting Started

To run this Next.js project locally on your machine:

### Prerequisites

You need to have **Node.js** (v18 or higher) and npm installed on your system.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/theoitssurabaya/webPorto.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd webPorto
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Run the development server:**
   ```bash
   npm run dev
   ```
5. **Open the project:**
   Open [http://localhost:3000](http://localhost:3000) in your browser to view the portfolio.
