# Theo Kawalisa Pinem - Professional Web Portfolio

Welcome to the repository for my professional web portfolio! This website serves as a digital showcase of my academic journey, professional experiences, and technical projects, with a strong focus on Computer Engineering, IoT, AI, and Robotics.

## 🌟 Overview

This portfolio is designed to provide a comprehensive view of my skills and achievements. It features a modern, responsive, and dynamic user interface with smooth animations and interactive elements to ensure an engaging user experience.

**Live Website:** [Insert link if deployed, e.g., theoitssurabaya.github.io/webPorto]

## ✨ Features

- **Modern & Dynamic UI/UX:** A vibrant light mode aesthetic with animated gradient backgrounds, glassmorphism elements, and smooth scrolling.
- **Interactive Elements:** Custom cursor, hover effects, and a tech stack ticker to bring the design to life.
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile viewing.
- **Structured Sections:**
  - **Profile:** An introduction to my background and core interests.
  - **Skills:** Categorized technical competencies in AI/ML, Embedded Systems/IoT, and Software Engineering.
  - **Education:** My academic timeline at Institut Teknologi Sepuluh Nopember (ITS).
  - **Experience & Certifications:** A timeline of my professional roles, leadership positions, and industry certifications.
  - **Projects:** Featured engineering works linking to respective repositories.
  - **Contact:** Links to my GitHub, LinkedIn, and email for easy communication.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and content organization.
- **CSS3:** Custom styling, animations, flexbox/grid layouts, and responsive media queries.
- **JavaScript (Vanilla):** DOM manipulation, custom cursor logic, scroll events, and interactive functionalities.
- **AOS (Animate On Scroll):** Library used for scroll-triggered animations to enhance visual appeal.
- **Google Fonts:** Utilizing 'Outfit' for modern headings and 'Inter' for clean body text.

## 🚀 Getting Started

To view this project locally on your machine, follow these simple steps:

### Prerequisites

You only need a modern web browser (Chrome, Firefox, Safari, Edge) to run this project. No complex backend or build tools are required.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/theoitssurabaya/webPorto.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd webPorto
   ```
3. **Open the project:**
   Simply open the `index.html` file in your preferred web browser. You can double-click the file or use a local development server like the "Live Server" extension in VS Code.
